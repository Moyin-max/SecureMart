# SecureMart One-Page Report

## Project Title

**SecureMart - Smart AI Security System for Convenience Stores**

## Introduction

SecureMart is a smart retail security solution designed to help convenience store owners protect their stores from theft, unauthorized access, and delayed incident response. The system demonstrates how motion sensing, AI-based suspicious activity detection, remote monitoring, and smart door control can work together in one connected platform.

## Problem

Many small stores rely only on traditional CCTV, which records incidents but does not actively respond to them. This creates challenges such as:

- shoplifting
- break-ins after store closing
- lack of real-time alerts
- slow manual response

## Solution

SecureMart provides a dashboard-driven security system that:

- detects store activity
- flags suspicious behavior
- locks the door automatically when needed
- keeps the alarm armed during risk situations
- sends alerts to the owner
- allows remote response from a phone or laptop

## System Components

- Motion detection logic
- Smart camera simulation
- Door lock control
- Alarm control
- Cloud-connected web dashboard
- Backend API for dashboard actions

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Render
- GitHub

## Key Features

- Live dashboard interface
- Motion and AI event simulation
- API-based backend communication
- Lock and unlock control
- Alarm arm and disarm control
- Incident logging
- Demo reset and automatic presentation mode

## Outcome

SecureMart successfully demonstrates a practical smart security workflow for convenience stores. It presents a realistic model of how IoT devices and cloud-based monitoring can improve retail safety and owner response time.
